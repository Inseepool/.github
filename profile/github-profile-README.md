# Inseepool

> 흩어진 이커머스 데이터를 하나로 모아, 셀러가 바로 의사결정할 수 있게.

카페24 기반 이커머스 셀러를 위한 **멀티 플랫폼 광고 성과 통합 분석 도구**입니다.

## 해결하는 문제

매일 아침 카페24, Meta 광고, 네이버 광고, Google 광고를 하나하나 열어서 엑셀에 수작업으로 정리하는 시간을 줄여줍니다.

## 레포지토리

| 레포 | 설명 |
|------|------|
| [inseepool](https://github.com/Inseepool/inseepool) | 메인 코드 — 데이터 수집, 리포트 생성, API, 대시보드 |
| [docs](https://github.com/Inseepool/docs) | 사용 가이드, API 키 발급 방법, 셀러 온보딩 문서 |

## 지원 플랫폼

- ☑ 카페24 (매출/주문)
- ☑ Meta Ads (인스타그램/페이스북 광고)
- ☑ 네이버 검색광고
- ☐ Google Ads
- ☐ TikTok Ads
